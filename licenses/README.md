# Included audio licenses

The files under `licenses/kenney/` are exact license notices copied from only the Kenney source packs represented in this repository's selected development catalog.

Every corresponding asset is identified as `CC0-1.0` in `manifest.json`. Attribution is not required by CC0, but the project credits Kenney in the manifest and release documentation.

The presence of a license notice does not mean the complete upstream pack or the complete Kenney collection is redistributed here. Only explicitly selected files listed in the manifest are included.
